Библиотека useful — руководство программиста

Введение

Библиотека useful (или "libuseful"), как следует из ее названия, является сборником всяких «полезностей», могущих в принципе использоваться и вне CX, и даже никак к CX не привязанных (кроме как местоположением своих исходных текстов).

Состав библиотеки:

cxlogger
Модуль работы с log-файлами.
cxnetacl
ACL...
cxscheduler
Планировщик на основе select(), по функциям аналогичный планировщику из Xt.
fdiolib
Модуль структурного ввода/вывода через файловые дескрипторы.
paramstr_parser
Разборщик параметров из текстового описания в бинарный вид.
seqexecauto
«Автомат последовательного исполнения»
timeval_utils
Набор функций для манипуляции со struct timeval.

(Пусть никого не смущают префиксы "cx", имеющиеся у трех компонентов — они введены просто «для порядку» — чтобы сделать имена уникальными.)
